# Project Overview

A curated collection of Banking & Fintech Analytics Projects demonstrating skills in data cleaning, modeling, visualization, and business insight generation.

In fintech, every transaction is both a data point and a decision. Behind a single declined payment sits a customer's trust, a fraud team's workload, and a finance team's revenue line all pulling at the same dataset from different directions. These projects sit inside that tension: digging through transaction-level data to find where risk concentrates, where revenue quietly leaks, and where a "normal" failure rate is actually a signal that something upstream is broken.

---

## 💼 About This Portfolio

This portfolio demonstrates my ability to:

Translate transaction-level data into risk, revenue, and compliance narratives that different stakeholders can act on
Spot the difference between a genuine risk pattern and a data or pipeline defect masquerading as one

### Zephyr Bank Transactional Report
This project analyzes 1,500 digital transactions for Zephyr Bank, a UK-based fintech neobank serving over 20,000 active customers entirely through mobile app, web, and automated payment rails. The dataset spans the H1 2026 review window (January–May 2026) and covers three interlinked dimensions - customers, transaction types, and merchant categories, feeding a single fact table of transaction-level detail
